    Reporting Issues

    If you find any bad or outdated keys in the provided list, you can create an issue on GitHub here. This helps ensure the keys are accurate and up to date.

    Theme Validator

    The validator uses JsonLint to check for syntax errors and format your theme JSON. Specific checks for themes have also been added to identify potential issues. If you find that the validator did not catch an error or flagged something incorrectly, you can create an issue on GitHub here. Note that the validator does not detect duplicate keys, but these duplicates will be automatically removed when you click "Format JSON."
